L1 to L2 Bridge Tutorial

This tutorial is intended to deepen the developer's understanding about L1 to L2 bridge. Even though we provide UI, we highly recommend reviewing the code for a deeper understanding. Additionally, we highly recommend developing your own SDK to embed with your game; we only provide basic functionality equipped SDK.

Before proceeding with the tutorial, please deploy ERC20 and ERC721 contracts on both L1 and L2 following the instructions in the contract folder.
